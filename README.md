## What AI Chat RTL Fixer Does

AI Chat RTL Fixer is made for Windows users who use desktop AI chat clients and need clearer handling for RTL languages like Persian, Arabic, Hebrew, and Urdu. It targets the chat area so the message content reads more naturally without altering the rest of the interface.

The app lives in the system tray and applies its behavior only while it is running, so the changes are runtime-based rather than permanent. It also ships with the Vazirmatn font and follows a privacy-first design with no telemetry and no external network calls.

---

## Key Capabilities

- Refines RTL text rendering in AI desktop chat apps
- Leaves code blocks, commands, file paths, URLs, and English text in left-to-right order
- Limits its changes to the chat area instead of affecting the sidebar or other UI parts
- Uses runtime-only adjustments that end when the app is disabled or closed
- Bundles the Vazirmatn font for improved RTL text appearance
- Designed for Windows 10 and Windows 11
- Offered in both installer and portable formats
- Works without telemetry or external network requests

## FAQ

**Will it modify the entire UI?**  
No. It is designed to affect the chat surface only, while sidebars and other areas stay unchanged.

**Is it limited to one language?**  
No. It is intended for RTL workflows that include Persian, Arabic, Hebrew, and Urdu text.

**Does it preserve technical text direction?**  
Yes. Code, commands, paths, URLs, and English text remain left to right.

**How do I update to a newer version?**

**What if a specific chat app does not work?**
